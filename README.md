# ギル
A Discord bot written in Node.js using [discord.js-commando](https://github.com/Gawdl3y/discord.js-commando)

## Installation

```bash
apt-get install ffmpeg  # or apt-get install libav-tools

git clone git@github.com:m5r/giru.git

cd giru && npm install
```

## Usage

```bash
npm run build
npm start
```
